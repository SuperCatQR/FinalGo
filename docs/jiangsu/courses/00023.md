### 考试范围

高等数学（工本）面向工科本科段自学考试，重点训练一元函数微积分、多元函数微积分、无穷级数、常微分方程和线性代数初步的基本概念、基本计算和工程应用能力。课程目标不是追求抽象证明的完整体系，而是要求考生能把“函数—极限—导数—积分—级数—方程—矩阵”串成可计算、可建模、可检查的工具链。

本页按工科高等数学通行顺序，将考试范围整理为八个模块：

| 模块 | 核心内容 | 学习目标 |
| --- | --- | --- |
| 函数、极限与连续 | 函数表示、复合函数、反函数、数列与函数极限、无穷小与无穷大、连续性、间断点 | 建立微积分语言，能判断极限存在性与连续性 |
| 一元函数微分学 | 导数定义、求导法则、高阶导数、微分、中值定理、洛必达法则、函数单调性与极值 | 用导数刻画变化率、最优化和曲线形态 |
| 一元函数积分学 | 不定积分、定积分、换元积分、分部积分、广义积分、几何与物理应用 | 用积分刻画累积量、面积、体积和平均效应 |
| 向量代数与空间解析几何 | 向量运算、平面与直线、二次曲面基本识别 | 为多元函数微积分和工程空间问题打基础 |
| 多元函数微分学 | 偏导数、全微分、复合函数求导、隐函数求导、极值与条件极值 | 处理多变量函数的局部变化和优化问题 |
| 多元函数积分学 | 二重积分、三重积分、曲线积分/曲面积分的基础口径（以教材附大纲为准） | 计算区域、体积、质量、重心等多维累积量 |
| 无穷级数 | 常数项级数、幂级数、泰勒展开、傅里叶级数初步（如大纲列入） | 判断收敛性，并用级数近似函数 |
| 常微分方程与线性代数初步 | 一阶微分方程、二阶常系数线性微分方程、行列式、矩阵、线性方程组 | 建立动态模型和线性计算能力 |

### 第一章 函数、极限与连续

| 编号 | 知识点 | 考核层级 | 审查重点 |
| ---: | --- | --- | --- |
| 1 | 函数概念、定义域、值域、函数表示法 | 识记/理解 | 注意实际问题中定义域的限制 |
| 2 | 单调性、奇偶性、周期性、有界性 | 理解/应用 | 会用图像和解析式判断性质 |
| 3 | 反函数、复合函数、初等函数 | 理解/应用 | 复合层次与定义域同步检查 |
| 4 | 数列极限与函数极限 | 理解 | 区分自变量趋向有限点与无穷远 |
| 5 | 左右极限、极限存在的充要条件 | 应用 | 分段函数和绝对值函数常考 |
| 6 | 无穷小、无穷大及其阶的比较 | 应用 | 等价无穷小替换须满足乘除结构 |
| 7 | 两个重要极限 | 应用 | 识别标准形式并做代换 |
| 8 | 函数连续性、间断点分类 | 应用 | 可去、跳跃、无穷间断点要分清 |

学习本章时，考生应把“极限存在”与“函数值存在”分开。函数在某点连续要求三件事同时成立：函数值存在、极限存在、二者相等。若只知道左右极限都存在但不相等，则极限不存在；若极限存在但函数值未定义或不相等，则是可去间断；若左右极限有限但不相等，则是跳跃间断。

本章是全课计算核心。常见失分不是公式不会，而是条件漏检：使用中值定理前没有说明连续与可导；使用洛必达法则前没有确认是 `0/0` 或 `∞/∞` 型；求最值时只求驻点却忘记端点；求复合函数导数时漏掉内层函数的导数。

## 题型与答题模板

> 本区块 ⭐✋——以下模板为原创解题方法论，不绑定第三方真题题号或参数；公式与例题计算需 PR 审查团队复核。

### 模板一：极限计算题

**题型特征**：题目要求求 `lim f(x)`，常见形式包括有理式、根式、三角函数、指数对数、分段函数或含参数表达式。若直接代入出现 `0/0`、`∞/∞`、`∞-∞`、`0·∞` 等未定式，需要先化简。

**答题结构**：

1. 先判断自变量趋向：有限点、无穷远、左极限或右极限。
2. 直接代入，识别是否为未定式。
3. 根据结构选择方法：因式分解、有理化、等价无穷小、变量代换、洛必达法则或夹逼定理。
4. 写出关键变形依据，最后给出极限值。

**评分要点**：方法选择正确；等价无穷小替换只用于乘除结构；洛必达法则使用前确认条件；分段函数必须分别算左右极限。

**完整示例（原创）**：求 `lim_{x→0} (sqrt(1+2x)-1)/x`。

解：直接代入得 `0/0` 型。对分子有理化：

`(sqrt(1+2x)-1)/x = [(sqrt(1+2x)-1)(sqrt(1+2x)+1)]/[x(sqrt(1+2x)+1)] = 2/[sqrt(1+2x)+1]`。

令 `x→0`，得到 `2/(1+1)=1`。因此极限为 `1`。

### 模板二：导数与函数性质题

**题型特征**：题目给出函数，要求求导、求切线方程、判断单调区间、极值、凹凸区间或拐点。

**答题结构**：

1. 写出函数定义域。
2. 求一阶导数，解 `f'(x)=0`，并列出不可导点。
3. 用一阶导数符号表判断单调性和极值。
4. 如需凹凸性，求二阶导数，解 `f''(x)=0` 或不存在点，判断符号变化。
5. 如求切线方程，用 `y-y0=f'(x0)(x-x0)`。

**评分要点**：定义域不能漏；极值候选点包括驻点和不可导点；最值题还要比较端点；切线斜率必须在指定点求导数值。

**完整示例（原创）**：设 `f(x)=x^3-3x^2+2`，求单调区间和极值。

解：定义域为全体实数。`f'(x)=3x^2-6x=3x(x-2)`。令 `f'(x)=0` 得 `x=0` 或 `x=2`。当 `x<0` 时，`f'(x)>0`；当 `0<x<2` 时，`f'(x)<0`；当 `x>2` 时，`f'(x)>0`。故函数在 `(-∞,0)` 与 `(2,∞)` 上单调增加，在 `(0,2)` 上单调减少。`x=0` 处由增到减，为极大值点，`f(0)=2`；`x=2` 处由减到增，为极小值点，`f(2)=-2`。

### 模板三：积分计算题

**题型特征**：题目要求求不定积分或定积分，通常需要选择基本公式、换元法、分部积分法或先化简再积分。

**答题结构**：

1. 判断被积函数结构：复合函数、乘积、分式、三角函数还是根式。
2. 不定积分写出积分常数；定积分注意上下限。
3. 换元时写明变量替换和微分关系；定积分换元要同步改上下限。
4. 分部积分按 `∫u dv = uv - ∫v du` 组织。
5. 结果可求导或代回检查。

**评分要点**：不定积分漏 `C` 会扣分；定积分上下限不能反；极坐标和二重积分中面积元素不能漏。

**完整示例（原创）**：求 `∫ x e^x dx`。

解：取 `u=x`，`dv=e^x dx`，则 `du=dx`，`v=e^x`。由分部积分公式，

`∫ x e^x dx = x e^x - ∫ e^x dx = x e^x - e^x + C = (x-1)e^x + C`。

求导验证：`[(x-1)e^x]' = e^x + (x-1)e^x = x e^x`，结果正确。

### 模板四：二重积分应用题

**题型特征**：题目给出平面区域和被积函数，要求计算面积、体积、质量或平均值。

**答题结构**：

1. 画出或描述积分区域。
2. 判断用直角坐标还是极坐标。
3. 写出积分次序和上下限。
4. 计算内层积分，再计算外层积分。
5. 应用题写明结果单位和几何含义。

**评分要点**：区域边界必须准确；极坐标面积元素为 `r dr dθ`；改变积分次序时要重新确定边界。

**完整示例（原创）**：求单位圆上半圆区域 `D={(x,y)|x^2+y^2≤1, y≥0}` 的面积。

解：用极坐标，区域为 `0≤r≤1, 0≤θ≤π`。面积为

`∫_0^π ∫_0^1 r dr dθ = ∫_0^π [r^2/2]_0^1 dθ = ∫_0^π 1/2 dθ = π/2`。

因此上半单位圆面积为 `π/2`。

### 模板五：级数收敛判别题

**题型特征**：题目要求判断常数项级数或幂级数是否收敛，可能还要求求收敛半径和收敛域。

**答题结构**：

1. 判断是正项级数、交错级数、一般项级数还是幂级数。
2. 正项级数优先考虑比较、比值、根值、积分判别。
3. 交错级数检查项的绝对值是否单调递减且趋于 0。
4. 幂级数先求收敛半径，再单独检查端点。
5. 给出绝对收敛、条件收敛或发散结论。

**评分要点**：一般项趋于 0 只是收敛必要条件，不是充分条件；比值判别极限等于 1 时方法失效；幂级数端点必须单独讨论。

**完整示例（原创）**：判断级数 `∑_{n=1}^{∞} 1/n^2` 的收敛性。

解：这是正项级数，与 `p` 级数比较，`p=2>1`。根据 `p` 级数判别法，`∑ 1/n^p` 在 `p>1` 时收敛。因此该级数收敛。

### 模板六：微分方程题

**题型特征**：题目给出一阶或二阶常系数微分方程，要求求通解或满足初值条件的特解。

**答题结构**：

1. 识别方程类型：可分离变量、一阶线性、二阶常系数齐次或非齐次。
2. 选择对应方法：分离变量、积分因子、特征方程、待定系数法。
3. 求通解。
4. 若有初值条件，代入确定常数。
5. 简单检查解是否满足原方程。

**评分要点**：任意常数个数应等于方程阶数；初值条件不能漏；二阶方程特征根重复时解的形式要改写。

**完整示例（原创）**：求微分方程 `y'=2y` 的通解。

解：当 `y≠0` 时分离变量得 `dy/y = 2 dx`。两边积分，`ln|y|=2x+C`，故 `y=Ce^{2x}`。其中 `C` 为任意常数，且 `C=0` 时也包含零解。因此通解为 `y=Ce^{2x}`。

### 模板七：线性方程组题

**题型特征**：题目给出线性方程组或矩阵，要求判断解的情况、求解或求矩阵的秩。

**答题结构**：

1. 写出增广矩阵。
2. 进行初等行变换化为阶梯形。
3. 比较系数矩阵秩与增广矩阵秩。
4. 判断无解、唯一解或无穷多解。
5. 若有解，回代或引入自由变量写出解集。

**评分要点**：行变换要合法；不能用列变换改变方程组解集；自由变量个数等于未知量个数减去系数矩阵秩。

**完整示例（原创）**：方程组 `x+y=3, x-y=1`。

解：两式相加得 `2x=4`，故 `x=2`；代入第一式得 `y=1`。因此方程组有唯一解 `(2,1)`。若用矩阵法，系数矩阵行列式为 `-2≠0`，也可判断唯一解。

### 分模块解析提示

- **极限题**：重点看未定式类型。若是根式差，优先有理化；若是三角函数接近 0，考虑标准极限；若是指数对数组合，考虑取对数或等价无穷小；若左右极限不等，应明确写出“不存在”。
- **导数题**：复合函数求导要逐层展开，隐函数求导要把 `y` 看作 `x` 的函数。切线题必须先求点坐标和斜率。
- **积分题**：不定积分强调方法选择，定积分强调上下限。涉及几何区域时，先画图再设限。若结果出现负面积，应检查上下曲线或积分顺序。
- **级数题**：判别法不是越复杂越好。先看通项是否趋于 0，再看正项/交错/幂级数分类；幂级数的端点要单独带入原级数判断。
- **微分方程题**：识别类型比计算更重要。可分离变量方程先分离，一阶线性方程用积分因子，二阶常系数方程先写特征方程。
