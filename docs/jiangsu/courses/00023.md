### 考试范围

本课程考核考生对多元微积分基本概念、基本理论与基本运算的掌握，以及运用所学知识解决简单几何、物理与实际问题的能力。考试范围以当次教材附大纲为准，覆盖六章：

- **空间解析几何与向量代数**：向量运算，平面、直线、柱面、旋转面与常见二次曲面的方程及图形。
- **多元函数的微分学**：二元函数极限、连续、偏导数、全微分、方向导数与梯度；复合函数与隐函数求导；极值与最值（含拉格朗日乘数法的了解）。
- **重积分**：二重、三重积分的概念、性质、计算（直角/极/柱面坐标）与应用。
- **曲线积分与曲面积分**：两类曲线积分、两类曲面积分的计算，格林公式，平面曲线积分与路径无关的条件。
- **常微分方程**：三类一阶方程、三类可降阶二阶方程、二阶常系数线性齐次方程的求解。
- **无穷级数**：数项级数审敛法，幂级数收敛域，函数展开成泰勒级数，区间 [−π, π) 上的傅里叶级数。

## 题型与答题模板

> 本区块 ⭐✋——须人工总结，不可仅靠机器生成。本课程为**计算题主导**，下面给出 **4 类题型**（单项选择、计算题、证明题、应用题）的四要素模板（题型特征、答题结构、评分要点、完整示例），满足 AC6「≥ 3 种题型，每种四要素齐全」。示例均为**原创演示用例**，非任何真题或样卷原题，仅示范解题方法论。

### 题型一：单项选择题

- **题型特征**：给定四个选项，考查概念辨析、定理条件、简单计算或判断（如向量平行/垂直、级数敛散、方程类型识别）。
- **答题结构**：① 判断该题考查的知识点范畴 → ② 优先用定义/定理快速排除明显错误项 → ③ 必要时代入特值或反例验证 → ④ 对计算型选择题直接算到可比对的形式 → ⑤ 锁定唯一符合项。
- **评分要点**：答案唯一；选择题不倒扣，无把握时也应基于排除法择优作答，不留空。
- **完整示例**（原创演示）：
  - 题：与向量 **a** = (1, 2, −2) 平行的单位向量是？
  - 思路：|**a**| = √(1+4+4) = 3，单位向量为 ±(1/3, 2/3, −2/3)。
  - 答案：(1/3, 2/3, −2/3) 或其反向 (−1/3, −2/3, 2/3)。

### 题型二：计算题（主体题型）

- **题型特征**：占分最高（样卷 60%）。要求完成一项明确的运算：求偏导/全微分、求重积分、求曲线/曲面积分、解微分方程、求级数收敛域或和函数等。
- **答题结构**：① 识别题型与适用方法（如积分先判坐标系、方程先辨类型）→ ② 写出关键公式/法则 → ③ 分步规范运算，保留必要中间步骤 → ④ 注意定义域、积分限、常数项 C 等细节 → ⑤ 回代或量纲/特值校验结果。
- **评分要点**：按步给分，方法正确即得过程分；最终结果需化简到位；漏写积分常数、积分限错误、符号错误为常见失分点。
- **完整示例**（原创演示）：
  - 题：设 z = x²y + sin(xy)，求 ∂z/∂x。
  - 思路：对 x 求偏导，y 视为常数；sin(xy) 用链式法则。
  - 解：∂z/∂x = 2xy + y·cos(xy)。

### 题型三：证明题

- **题型特征**：要求验证某函数是给定方程的解、证明级数敛散、证明等式或某性质成立（常出现在计算题/综合题中“验证…”形式）。
- **答题结构**：① 明确要证的命题与已知条件 → ② 选定方法（代入验证、定义法、用已知定理）→ ③ 由已知出发逐步推导，每步给出依据 → ④ 推到目标结论，写出“故…成立/得证”。
- **评分要点**：逻辑链完整、每步有据；验证类题须把待验对象代回原式并化简到恒等；不得跳步导致依据缺失。
- **完整示例**（原创演示）：
  - 题：验证 y = e^{2x} 是微分方程 y″ − 3y′ + 2y = 0 的解。
  - 思路：求 y′、y″ 代回方程，化简看是否恒为 0。
  - 解：y′ = 2e^{2x}，y″ = 4e^{2x}；代入得 4e^{2x} − 6e^{2x} + 2e^{2x} = 0，恒成立，故 y = e^{2x} 是该方程的解。

### 题型四：应用题

- **题型特征**：把几何/物理或实际情境转化为数学模型再求解，如用极值求最优、用重积分求体积/质量、用微分方程描述变化过程。
- **答题结构**：① 读题提取已知量与所求量 → ② 建立数学模型（设变量、列函数/方程、定区域）→ ③ 调用相应方法求解（极值条件、积分、解方程）→ ④ 结合实际取舍解（如舍去负值/非定义域解）→ ⑤ 回答实际问题并标明单位/含义。
- **评分要点**：建模正确是得分前提；模型对、计算误差只扣计算分；须回到实际情境作答，不能只停在数学结果。
- **完整示例**（原创演示）：
  - 题：求函数 f(x, y) = x² + y² 在约束 x + y = 2 下的最小值。
  - 思路：条件极值，用拉格朗日乘数法或代入消元。代入 y = 2 − x，得 g(x) = x² + (2−x)²，g′(x) = 4x − 4 = 0 ⟹ x = 1。
  - 解：x = 1, y = 1，最小值 f = 2。
