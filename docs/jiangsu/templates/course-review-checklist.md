# 课程页审核清单 v2

> 来源：PRD §9，共 20 项。适用于课程页从 🟡 机器初稿 → 🟢 人工校对可发布 的逐项检查。
> 关联 AC：AC12（通过审核清单全部检查项）
>
> 四项维度：**内容准确性**（8 项）→ **版权合规**（4 项）→ **格式规范**（6 项）→ **链接有效性**（2 项）

---

## 一、内容准确性（CHK-01 ~ CHK-08）

- [ ] CHK-01 课程代码、名称与官方专业计划一致
- [ ] CHK-02 学分与官方计划一致
- [ ] CHK-03 教材版本为当前考期有效版本
- [ ] CHK-04 章节知识树与考纲目录结构一致，知识点覆盖率 ≥ 90%（分母按 AC4 定义：考纲正文中明确列出的知识点条目数）
- [ ] CHK-05 高频概念无凭空编造，考频排序有真题统计依据
- [ ] CHK-06 过渡期特殊说明准确
- [ ] CHK-07 「政策与时效」节内容准确
- [ ] CHK-08 所有日期使用实际日期非占位符

## 二、版权合规（CHK-09 ~ CHK-12）

- [ ] CHK-09 真题解析不包含第三方原题全文（版权合规）
- [ ] CHK-10 解题思路描述通用方法论，不绑定具体考题题号和参数（版权合规 v2 强化）
- [ ] CHK-11 第三方来源使用声明已包含（「基于公开考试信息的原创分析」）
- [ ] CHK-12 AI 生成内容已标注且经人工校对，校对签名可追溯至 Git commit

## 三、格式规范（CHK-13 ~ CHK-18）

- [ ] CHK-13 题型模板包含四个完整要素（题型特征、答题结构、评分要点、完整示例），示例不绑定具体考题
- [ ] CHK-14 新旧代码顶替关系使用标准 4 字段格式（新代码/旧代码/起始考期/过渡后定位）
- [ ] CHK-15 「来源与引用」节格式与 major 模板一致
- [ ] CHK-16 旧代码页面有醒目跳转链接（03708→15043, 03709→15044）
- [ ] CHK-17 面包屑导航可用
- [ ] CHK-18 Markdown 格式规范，打印可读

## 四、链接有效性（CHK-19 ~ CHK-20）

- [ ] CHK-19 考纲链接可访问或本地 PDF 路径有效（「来源与引用」节）
- [ ] CHK-20 适用专业清单完整（公共课），来源标注自动生成时间戳

---

## 使用说明

1. **逐项检查**：按 CHK-01 → CHK-20 顺序逐项核对，勾选通过项 `[x]`。
2. **不通过处理**：未通过项在行末标注原因和修正建议，交由内容建设 Agent 修正后复检。
3. **全量通过**：20 项全部 `[x]` 后，课程页可标记为 🟢 已验证可发布。
4. **部分通过**：若仅链接有效性或格式规范项未通过，可标示为 🟡 待修正；内容准确性或版权合规任一项未通过，维持 🔴 或 🟡 不可发布。
5. **版本关联**：本清单基于 `templates/course.md` v2 模板设计，模板升级时需同步更新。
